# DataGalaxy（数据星河）

从 0 到 1，手把手教你构建属于自己的数据可视化大屏。

## 项目简介

DataGalaxy 是一个面向数据可视化初学者的开源项目。无论你是学生、前端开发者，还是数据分析师，只要你对数据大屏感兴趣，这个项目都能帮你快速上手。

### 你将从中学到什么

- 数据大屏的整体架构设计思路
- 前端可视化库（如 ECharts、D3.js）的实战使用
- 大屏适配与响应式布局方案
- 动态数据接入与实时刷新机制
- 从项目搭建到最终部署的完整流程

### 适合谁看

- 在校学生：课程设计、毕业设计、竞赛作品
- 前端新人：积累一个拿得出手的项目经验
- 数据分析师：将分析结果以可视化大屏形式呈现

## 快速开始

```bash
# 克隆项目
git clone https://github.com/wjy7715/DataGalaxy.git

# 进入目录
cd DataGalaxy

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
